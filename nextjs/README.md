# Development

```properties
npm run build

```

# Deploy

## SST - needs linux / ubuntu

- npx create-sst@2.23.4
  - then run the following
    - npm i
    - npx sst dev
      - stage - stagestagw
    - npm run dev
- deploy
  - npx sst deploy --stage prod
